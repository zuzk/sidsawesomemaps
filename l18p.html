Map Instillation Instructions

 

#1 Download the map you want to install

 

#2 Make sure you have Minecraft installed

 

#3 Extract the map by right clicking it and selecting "Extract Here"

(If this option is not available for you then  you need to install WinRAR

(By clicking the WinRAR above you will be taken to the WinRAR page

all you need to do from there is install the proper version for your PC)

 

#4 Put the folder that was extracted in a easy place to find

 

#5 Press the "Windows" key and hold it down, while you are holding it down press the "R" key (this should pop up run)

#6  In Run type %AppData% then press the "enter" key   (this should pull up your application data folder)

 

#7 In your application data folder find the folder called ".minecraft"  and double click on it

 

#8 From there find the folder called "saves" then put the folder you extracted earlier

into the "saves" folder.

 

#9 Launch up Minecraft and play in the appropriate version that the map needs

(If you don't know how to pick the appropriate version then click here for help)  

 

#10 Your done!
